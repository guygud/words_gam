# Тетрис со словом дня

Игра-тетрис, где линии сгорают только при наличии корректных слов из словаря.

## Запуск игры

Из-за ограничений браузера для ES6 модулей, игру нужно запускать через локальный HTTP сервер.

### Вариант 1: Python (рекомендуется)

Если у вас установлен Python 3:

```bash
# Python 3
python3 -m http.server 8000

# Или Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 2: Node.js

Если у вас установлен Node.js:

```bash
# Установите http-server глобально (один раз)
npm install -g http-server

# Запустите сервер
http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 3: PHP

Если у вас установлен PHP:

```bash
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

## Управление

- **← →** - движение фигуры влево/вправо
- **↓** - ускоренное падение
- **Space** - поворот фигуры

## Правила игры

1. Фигуры падают на поле 10×20
2. Каждая клетка фигуры содержит букву из слова дня
3. Линии сгорают только если в них найдено хотя бы одно слово из словаря (минимум 3 буквы)
4. За каждое найденное слово начисляются очки
5. Если найдено слово дня целиком - дополнительный бонус!

## Структура проекта

```
/js/
  config.js              # Конфигурация игры
  models/                 # Модели данных
  services/               # Сервисы (словарь, проверка слов, очки)
  core/                   # Основная логика игры
  main.js                 # Точка входа
```
